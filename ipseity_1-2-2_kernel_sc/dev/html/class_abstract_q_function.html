Detailed Description

Abstract Q-Function.

The aim of a learning agent is to build a policy, that is a function mapping states to actions. A way to implicitly build this policy is to store for each stimulus-response pair (x,u) a Q-value Q(x,u) which represents the expected cumulative rewards received in stimulus x when carrying out response u and to take for a stimulus x the response u* that maximises Q(x,u') over all the possible responses u' in x. A Q-Function class provides methods to access and updates these Q-values.
