Detailed Description

Abstract Learning Module.

A Learning Module implements the methods used by an agent to learn from interactions with its environment. An interaction is of the form <s,a,ns,r>, where s is the current state, a is the action carried out in state s, ns is the state reached after carrying out action a and r is the received reward from transition <s,a,ns>.
